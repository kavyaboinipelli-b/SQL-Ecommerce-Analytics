SQL E-Commerce Analytics

Project Overview

This project analyzes e-commerce sales data using SQL to answer real business questions and generate actionable insights.

The goal is to demonstrate SQL skills commonly required for Data Analyst roles.

---

Skills Demonstrated

- Joins
- Subqueries
- CTEs
- Window Functions
- Aggregations
- CASE Statements
- Views
- Date Functions
- KPI Analysis

---

Business Questions

1. Top 10 customers by revenue
2. Best-selling products
3. Monthly sales trends
4. Regional performance analysis
5. Revenue growth analysis
6. Customer retention insights

Tools Used

- MySQL
- SQL
- GitHub
